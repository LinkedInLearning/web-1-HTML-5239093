Der Einstieg in HTML und CSS

Webseiten bestehen am Bildschirm aus rechteckigen Kästchen. Beim Umgang mit diesen Kästchen haben HTML und CSS klar getrennte Aufgaben:

  • HTML-Elemente strukturieren eine Webseite und erstellen die Kästchen.
  • CSS-Regeln gestalten die Kästchen und deren Inhalt.

Beide Sprachen arbeiten bei der Erstellung und Gestaltung eng zusammen. Weitere Informationen finden Sie auf der Website zum Kurs.
